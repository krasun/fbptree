A persistent storage (in file) based using [B+ tree](https://en.wikipedia.org/wiki/B%2B_tree) with byte-slice keys and values. 

## Installation 

To install, run:

```
go get github.com/krasun/fbptree
```

## Tests

Run tests with: 

```
$ go test .
ok  	github.com/krasun/fbptree	0.679s
```
